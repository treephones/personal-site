Anagrams Bot

A bot which completely automates the iMessage game, Anagrams, by taking a screenshot of the phone screen, parsing the text from the screenshot, generating all possible english words from the letter set, and sending coordinates to an arduino which taps the letter on the iPhone.

  • Python (OpenCV, Numpy, Tesseract, Serial)
  • C (Arduino)

CroPixelBot

A versatile discord bot with multiple functions, including a friends list tracker, which tracks the friends list and other stats of specified users on the Hypixel minecraft server via periodic requests to the Mojang + Hypixel REST API's, text to speech voice channel output for people without mics, and image editing.

  • Java
  • Java Discord API (JDA)
